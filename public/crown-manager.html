How it Works

1. Select a base Ollama model (e.g., llama2, mistral)

2. Choose a Crown with your knowledge/lore/data

3. System generates an Ollama Modelfile

4. Run: ollama create new-model -f modelfile

5. Your fine-tuned model is ready!
